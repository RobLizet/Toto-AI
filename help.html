⏱️ Automatische Scan BIJGEWERKT v20
HOE WERKT DE AUTO SCAN?
Inschakelen via Instellingen → ⏱️ Automatische Scan.

De app scant elke uur automatisch alle wedstrijden van vandaag op value — ook als de app op de achtergrond staat.

Configureerbaar:
• Scanvenster van/tot — standaard 14:00–18:00
• Sla vandaag over — scan overslaan voor deze dag
• Start nu — direct een scan uitvoeren
WAT DOET DE SCAN?
1. Laadt wedstrijden van vandaag uit alle competities
2. Berekent Poisson kansen + AI analyse (blend 65/35)
3. Filtert picks boven de value drempel
4. Slaat picks op in Dashboard → Auto-scan picks
5. Logt elke scan in de Scan Log (Analyse tab)
6. Stuurt een push notificatie bij sterke value (als notificaties aan staan)
ℹ️ Vereist: Anthropic API key in Instellingen voor de AI analyse stap. Zonder key draait alleen het Poisson model.
⚡ Value Scan
HOE WERKT DE VALUE SCAN?
De scan analyseert alle geladen wedstrijden tegelijk:

1. Haalt H2H, vorm, blessures en statistieken op
2. Berekent kansen via het Poisson model
3. AI analyseert context en geeft eigen kansinschatting
4. Blend: 65% Poisson + 35% AI = gecombineerde kans
5. Vergelijkt gecombineerde kans met bookmaker quote (incl. vig-correctie)
6. Berekent value% — hoe veel de kans hoger is dan de quote impliceert
7. Pick wordt automatisch opgeslagen in de Scan Log
💡 Drempel: Gebruik alleen picks met ≥10% value én confidence ≥7. Dat zijn de signalen met de meeste onderbouwing.
🏆 TRIPLE / DOUBLE LOCK
De sterkste picks krijgen een Lock badge:

🏆 TRIPLE LOCK Value ≥15% + conf ≥8 + Poisson bevestigt
🔒 DOUBLE LOCK Value ≥12% + conf ≥7

Lock picks worden automatisch opgeslagen in de Backtest tab én de Scan Log.
VIG-CORRECTIE
Bookmakers bouwen een marge (vig) van ~4-6% in hun quotes. Zonder correctie lijkt bijna alles value. De app berekent de overround (1/o1 + 1/oX + 1/o2) en corrigeert de fair odds hierop. Daardoor zijn de value-percentages realistischer.
📐 Poisson Model
WAT IS HET POISSON MODEL?
Poisson is een wiskundig model dat de kans berekent op een bepaald aantal doelpunten, gebaseerd op de aanvals- en verdedigingssterkte van beide teams.

De app gebruikt:
• Gemiddelde doelpunten gescoord thuis/uit (seizoen)
• Gemiddelde doelpunten tegen thuis/uit
• Recente vorm — laatste 5 wedstrijden tellen voor 40%
• xG data als beschikbaar
• Competitiespecifiek gemiddelde (Eredivisie ≠ Serie A)
DIXON-COLES CORRECTIE
Standaard Poisson onderschat lage scores (0-0, 1-0, 0-1, 1-1). De app past een Dixon-Coles tau correctie toe die de kansen op lage scores verhoogt naar wat historisch echt voorkomt.

Na correctie worden alle kansen hergenormaliseerd tot 100%.

1

X
Gelijkspel — beide teams evenveel goals.
2
Uitteam wint — meer goals dan thuisteam.
OVER/ONDER

BTTS-J
Ja — beide teams scoren minstens 1 goal.
BTTS-N
Nee — minstens één team houdt een clean sheet.

1X

X2

💰 Quotes & Kansen
WAT IS EEN QUOTE?
Een quote geeft aan hoeveel je terugkrijgt per ingezette euro.

Quote 2.00 → €10 inzet = €20 terug (€10 winst)
Quote 1.50 → €10 inzet = €15 terug (€5 winst)
Quote 3.50 → €10 inzet = €35 terug (€25 winst)

Quote 1.20 = grote favoriet  ·  Quote 5.00 = outsider
VALUE BET
Een value bet is een weddenschap waarbij jij denkt dat de echte kans hoger is dan de bookmaker inschat.

Voorbeeld: Quote 3.00 op gelijkspel (= 33% kans). Als de AI+Poisson 45% berekent → value bet van +12%.
💡 Tip: Op lange termijn win je alleen structureel als je consequent value bets vindt. Eén gewonnen bet zegt niets.
📊 Statistieken & ROI
HITRATE
Percentage gewonnen weddenschappen. Hitrate 60% → 6 van 10 gewonnen.

Let op: een hoge hitrate zegt niet alles. Bij lage quotes (1.30) win je vaak maar verdien je weinig. Kijk altijd naar ROI.
ROI — RETURN ON INVESTMENT
ROI% = (winst − verlies) / totaal ingezet × 100

+10% ROI → €10 winst per €100 ingezet
−15% ROI → €15 verlies per €100 ingezet

Professionele tipsters halen gemiddeld 5-10% ROI. Dat klinkt laag maar is uitstekend.
VALUE KALIBRATIE
De Scan Log toont de value kalibratie per bucket (0-10%, 10-20%, 20-30%, 30%+).

Een goed gekalibreerd model: picks met 20% value winnen ook ~20% vaker dan de quote impliceert. Als alle buckets vergelijkbaar scoren, werkt de value berekening niet goed genoeg.

🧠 v20 Intelligence Core NIEUW v20
CONFIDENCE ENGINE
Elke pick krijgt een confidenceFinal score (0–100) berekend door de v20 Confidence Engine. Deze houdt rekening met:

• League factor — betrouwbaarheid van data per competitie
• Odds range — picks buiten 1.60–4.50 krijgen straf
• Markt factor — 1X2 betrouwbaarder dan exotische markten
• Data kwaliteit — hoeveel statistieken beschikbaar zijn
• Calibratiefactor — historische prestatie per league
⭐ ELITE PICKS
De sterkste picks worden gemarkeerd als Elite:

⭐ ELITE ConfidenceFinal ≥72 + Value ≥8% + Odds 1.60–4.50

Elite picks krijgen prioriteit bij push notificaties en worden bovenaan de scan resultaten getoond.
📈 ZELFLEREND SYSTEEM
Na elke geverifieerde uitslag leert het systeem bij:

• League calibratie — de factor per competitie wordt bijgesteld op basis van historische hitrate en ROI
• CLV (Closing Line Value) — vergelijkt jouw odds met de sluitingskoers. Positieve CLV = je had een voordeel.
• Wekelijkse herberekening — elke zondag 06:00 worden alle league factoren opnieuw berekend op basis van alle historische data
ℹ️ Hoe meer picks, hoe beter het systeem wordt. Na 100 picks zijn de calibratiefactoren statistisch betrouwbaar.
📊 ODDS MOVEMENT
De worker slaat openingsodds op en vergelijkt ze met de huidige odds:

• Steam move — odds dalen snel = sharp money aan het werk
• Drift — odds stijgen = publiek geld op de andere kant

Een pick waarbij de odds bewegen in jouw richting is sterker.
