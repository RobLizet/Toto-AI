⚡ Wat is TOTO AI?

TOTO AI is een persoonlijke voetbalanalyse-app die value bets detecteert — wedstrijden waarbij de kans op winst hoger is dan de bookmaker inschat. De app combineert een Poisson statistisch model met AI-analyse om kansen te berekenen en vergelijkt die met de marktquotes.

⚠️ Fictief / educatief gebruik. Goк met beleid. 18+

🚀 Snel starten

  • 2Kies een competitie (bijv. Eredivisie) — wedstrijden laden automatisch
  • 3Tik ⚡ SCAN VALUE — de app analyseert alle wedstrijden en toont picks met value

📐 Hoe werkt de Value Scan?
⚡ Value formule
value% = (AI_kans × odds − 1) × 100
Voorbeeld: 60% kans × 2.10 odds = 1.26 → +26% value
⚡ ≥15% sterk ⚡ 5-15% licht 0-5% fair <0% geen value
```
📐 Poisson model

De scan haalt seizoensstatistieken op per team en berekent verwachte doelpunten (lambda). Hieruit volgen de winstkansen via een Poisson-verdeling — volledig wiskundig, zonder AI-bias.

λ_thuis = aanvalskracht × verdedigingszwakte tegenstander / competitiegemiddelde

Eindkans = 65% Poisson + 35% AI. De AI corrigeert alleen voor context: blessures, H2H, motivatie.

🟡 Middenvelder−4% aanval + verdediging
```

🎯 Wanneer inzetten?

Zet alleen in als drie van vijf groen zijn:

  • ✅Value ≥10% — liefst ≥15%
  • ✅Confidence ≥7/10 — genoeg data
  • ✅Poisson aanwezig (📐 P+AI badge)
  • ✅Odds 1.80—3.50 — sweet spot
  • ✅Reden verwijst naar concrete data — niet alleen "ranglijst"
Vier of vijf groen? Serieuze pick. Twee of minder? Overslaan.
⚠️ Value >55% is gecapped — te hoge waarden wijzen op datafout (verouderde stats, ontbrekende blessures). Controleer altijd handmatig.
💰 Half Kelly & Bankroll
Kelly formule
Kelly% = (kans × odds − 1) / (odds − 1) × 50%
Max 25% van bankroll per pick (ingebouwde cap)

Half Kelly geeft 75% van de groei van full Kelly met de helft van de variance. Professionele value-betters gebruiken altijd half Kelly of minder.

Praktische vuistregels
Kelly%Betekenis
0%Geen value — niet inzetten
1—3%Lichte value — kleine inzet
3—6%Goede value — normale inzet
>6%Sterk — maar controleer blessures

📊 Value Backtest Tracker

Elke value pick (≥5%) wordt automatisch opgeslagen in de Tracker tab. Na elke gespeelde wedstrijd tik je op 🔍 CHECK om de uitslag op te halen.

De tracker berekent dan automatisch:

  • →Hitrate — % gewonnen van afgeronde picks
  • →ROI — rendement op flat €1 inzet per pick
  • →Grafiek — cumulatief resultaat vs break-even lijn
Pas na ~50 picks zie je een betrouwbaar patroon. Na 200 picks weet je zeker of de scanner edge heeft.

```
Waarom vindt de scan soms niks?

Bookmakers zitten soms gewoon goed in de markt. Geen picks = geen value. Dat is goed nieuws — de scanner doet zijn werk door niet slechte picks te tonen.

Wat betekent "Fout: Overloaded"?

Anthropic's servers zijn tijdelijk druk. De app probeert automatisch 3x opnieuw (na 15s/30s/45s). Werkt het nog steeds niet? Probeer het later — 's avonds na 20:00 is rustiger.

Waarom staat Jacks niet in de bookmaker-lijst bij een wedstrijd?

Jacks is een Nederlandse bookmaker en zit niet in de API-Football database. Tik op "✏️ Andere quotes gebruiken" en vul de Jacks-quotes handmatig in. Jacks staat wel als standaard in de handmatige dropdown.

Wat is de beste combi-strategie?

Combineer alleen picks uit verschillende competities (onafhankelijk). Kies legs met odds 1.60—2.50. Maximaal 3 legs. Gebruik alleen picks die de value scan heeft goedgekeurd.

Klopt de hitrate van 100%?

Bij weinig afgeronde picks kan 1/1 = 100% geven. De app toont het sample size bij <10 picks: "100% (1)". Pas na 20+ afgeronde picks is de hitrate betekenisvol.
```
