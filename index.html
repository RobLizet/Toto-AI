⚡ WAT IS VALUE?

Value = wanneer de AI een hogere winstkans schat dan de bookmaker. Dan heb je op lange termijn statistisch voordeel.

value % = (AI_kans × quote − 1) × 100

Voorbeeld: AI zegt Ajax wint met 60% kans, bookmaker geeft quote 2.10

0,60 × 2,10 = 1,26
→ (1,26 − 1) × 100 = +26% value 🔥

Kleuren: ⚡ groen ≥15% sterk · oranje 5–15% licht · grijs 0–5% fair · rood <0% geen value.

📖 TOTO AI Gids

🚀 Aan de slag

Vul bij ⚙️ Instellingen je API keys in:

  • Anthropic — voor AI analyse & tips
  • API-Football — voor wedstrijden & statistieken

Daarna kun je wedstrijden laden door op een competitie te tikken.

🌍 Alle competities vandaag
Tik op de blauwe knop 🌍 ALLE COMPETITIES VANDAAG om in één keer alle wedstrijden van alle competities te laden, gegroepeerd per competitie.
🏆 WK 2026
De gouden WK 2026 chip onderaan de competitielijst toont de countdown. Wedstrijden worden beschikbaar vanaf 11 juni 2026.
⚽ Wedstrijden
Wedstrijden van vandaag t/m overmorgen per competitie. Tik op een wedstrijd voor analyse. Multi-scan: houd een competitie lang ingedrukt om meerdere tegelijk te scannen.
🤖 Analyse
Selecteer een wedstrijd en tik op ANALYSEER voor een volledige AI-analyse: kansen, vorm, statistieken, tactiek, risico en een bet advies.
⚡ Tips
AI-gegenereerde combi tips op basis van de geselecteerde competitie. Tik op een tip om hem toe te voegen aan je wallet.
💶 Wallet
Fictief geld bijhouden. Saldo, winst/verlies, hitrate en een grafiek per bron (Analyse / Combi / Eigen). Bets swipen voor snelle win/verlies invoer.
⚡ Backtest
Alle value scan picks worden automatisch opgeslagen. Bekijk resultaten per competitie via het 🏆 Per competitie sub-tab: wins/totaal + hitrate per competitie.
📒 Tracker
Bijhouden van echte weddenschappen. Filter op Alle / Open / Gewonnen / Verloren / Analyse / Combi. Streak en ROI per bron worden automatisch berekend.
⚙️ Instellingen
API keys, standaard inzet, notificaties, Triple Lock odds filter, auto donker thema en Firebase cloud backup.
⚡ Wat is value?
Value = de AI schat een hogere winstkans dan de bookmaker impliceert.
value% = (AI_kans × quote − 1) × 100
Kleuren:
🟢 Groen ≥15% — sterk value
🟡 Oranje 5–15% — licht value
⚫ Grijs 0–5% — fair geprijsd
🔴 Rood <0% — geen value
🏆 Triple Lock
Een wedstrijd heeft Triple Lock als hij aan alle 3 signalen voldoet:

✓ Value ≥5%
🎲 Confidence ≥7/10
📐 Poisson model bevestigt

Triple Lock picks verschijnen in een gouden sectie bovenaan de matchlijst.
🔑 Double Lock
Precies 2 van 3 signalen aanwezig. Verschijnt in een blauwe sectie onder Triple Lock, met badge welk signaal ontbreekt.
📐 Poisson model
Berekent verwachte doelpunten op basis van aanvals- en defensiestatistieken. Gecombineerd met AI geeft dit de meest betrouwbare kansinschatting.
½ Kelly Criterion
Optimale inzetgrootte op basis van je bankroll en de value. Kelly% × saldo = aanbevolen inzet. De app gebruikt halve Kelly voor minder risico.
💶 Fictief saldo
Het wallet systeem werkt met fictief geld. Standaard €500. Storten en opnemen via de knoppen. Reset zet alles terug naar beginstand.
👆 Swipe gestures
Op open bets in de wallet:
← Swipe links = ✗ Verlies
→ Swipe rechts = ✓ Win
Of tik op de status knop om te wisselen: Open → Win → Verlies → Open
🔍 Automatisch checken
De app checkt elke 15 minuten automatisch of open bets al gespeeld zijn (tussen 15:00 en 23:00). Of tik op Alles checken voor een handmatige check.
☁️ Cloud backup
Via Firebase worden wallet, tracker en backtest automatisch gesynchroniseerd. Bij ⚙️ Instellingen → Firebase kun je handmatig opslaan of herstellen.
📊 Grafiek per bron
De saldo grafiek kan worden gefilterd op bron: Alles / 🤖 Analyse / ⚡ Combi / ✏️ Eigen. Zo zie je welke strategie het meeste oplevert.
🔔 Push notificaties aanzetten
Ga naar ⚙️ Instellingen → Value Notificaties en tik op AANZETTEN. Geef toestemming als de browser daarom vraagt.
Wanneer krijg je een melding?
🏆 Triple Lock — na Value Scan
🔑 Double Lock — na Value Scan
✅ Bet gewonnen — bij auto-check
❌ Bet verloren — bij auto-check

Tik op de melding om direct naar de wedstrijd te gaan (vereist nieuwe sw.js).
🔕 Geen geluid?
Ga naar Telefooninstellingen → Apps → Chrome → Meldingen → zet geluid aan. Chrome-meldingen staan standaard op stil op Android.
🔔 Test notificatie
Bij ⚙️ Instellingen → Value Notificaties staat een 🔔 TEST knop. Gebruik die om te controleren of notificaties werken.
💡 Beste workflow
1. Tik op 🌍 Alle competities vandaag
2. Tik op ⚡ SCAN VALUE voor Triple/Double Lock picks
3. Analyseer interessante wedstrijden via 🤖 Analyse tab
4. Voeg toe aan 💶 Wallet of 📒 Tracker
5. De app checkt automatisch resultaten
🌙 Auto donker thema
Bij ⚙️ Instellingen → Auto donker thema. Schakelt automatisch naar donker om 20:00 en terug om 08:00.
🎯 Triple Lock odds filter
Bij ⚙️ Instellingen → Triple Lock filter kun je een minimale quote instellen. Standaard ≥1.60 — lage odds worden uitgefilterd.
📌 Multi-scan
Houd een competitie lang ingedrukt om hem toe te voegen aan multi-scan. Selecteer meerdere competities en tik dan op ⚡ SCAN ALLES.
⚡ Backtest begrijpen
ROI in de backtest is berekend op flat €1 inzet per pick — de eerlijkste maatstaf. Een ROI van +8% betekent €0.08 winst per €1 ingezet over alle picks.
