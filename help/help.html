⚡ Value Scan & Signalen
HOE WERKT DE VALUE SCAN?
De value scan combineert drie signalen per wedstrijd:

POISSON Wiskundige kansberekening op basis van teamstatistieken (goals voor/tegen, aanvalskracht, verdedigingszwakte)

AI ANALYSE Claude analyseert de data en vergelijkt zijn kansschatting met de boekmakersquote

STEAM Odds movement — als een quote daalt is dat een signaal dat professionele gokkers al inzetten

Een pick verschijnt alleen als minimaal 2 van 3 signalen overeenkomen.
🏆 TRIPLE LOCK — DOUBLE LOCK
🏆 TRIPLE LOCK Alle 3 signalen wijzen dezelfde kant op. Confidence 8+/10. Sterkste signaal.

🔒 DOUBLE LOCK 2 van 3 signalen kloppen. Confidence 6-7/10. Goed signaal.

Picks onder 7/10 confidence worden niet opgeslagen in de Backtest — die zijn te onzeker.
💡 Advies: Gebruik alleen Triple/Double Lock picks voor echte weddenschappen. Wacht op minimaal 3-5 van dit soort picks per week.

🎯 Confidence & Value
CONFIDENCE SCORE (1–10)
Hoe zeker is het model van de pick:

8–10 HOOG Poisson + AI + Steam allemaal eens. Triple Lock.

6–7 GEMIDDELD Minimaal 2 van 3 signalen kloppen. Double Lock.

1–5 LAAG Wordt niet opgeslagen. Te weinig zekerheid.
VALUE % — WAT BETEKENT HET?
Value % = hoeveel procent de AI denkt dat de bookmaker de kans onderschat.

Formule: (AI kans% × quote − 1) × 100

+21% value → AI schat 55% kans, bookmaker implied 34% → grote onderschatting
+5% value → kleine marge, maar nog steeds positief

Hoe hoger de value%, hoe groter het potentiële voordeel op lange termijn.
½ KELLY INZET
Kelly% is een wiskundige formule die de optimale inzet berekent op basis van de AI kans en de quote.

TOTO AI toont de halve Kelly (conservatiever, minder risico).

Kelly 9.3% bij €500 saldo → optimale inzet ≈ €46

Kelly 0% = geen value, beter niet inzetten.

💰 Quotes & Kansen
WAT IS EEN QUOTE?
Een quote geeft aan hoeveel je terugkrijgt per ingezette euro.

Quote 2.00 → €10 inzet = €20 terug (€10 winst)
Quote 1.50 → €10 inzet = €15 terug (€5 winst)
Quote 3.50 → €10 inzet = €35 terug (€25 winst)

Quotes onder 1.60 zijn grijs weergegeven in de app — te laag voor value betting.
IMPLICIETE KANS
Kans % = (1 / quote) × 100

Quote 2.00 → 50% kans volgens bookmaker
Quote 3.00 → 33% kans
Quote 4.00 → 25% kans

Als de AI een hogere kans inschat dan de bookmaker → value bet!
VALUE BET
Een value bet is een weddenschap waarbij de echte kans hoger is dan de bookmaker inschat.

Voorbeeld: Bookmaker geeft 3.00 op gelijkspel (33% kans implied). Poisson model berekent 45% kans → +35% value → Triple Lock kandidaat.
💡 Tip: Op lange termijn win je alleen structureel als je consequent value bets vindt. Eén gewonnen bet zegt weinig — kijk naar ROI over 50+ picks.
📊 Statistieken & ROI
ROI — RETURN ON INVESTMENT
ROI% = (winst − verlies) / totaal ingezet × 100

+10% ROI → Voor elke €100 ingezet maak je €10 winst
−15% ROI → Voor elke €100 ingezet verlies je €15

Professionele tipsters halen gemiddeld 5–10% ROI op lange termijn. Pas bij 50+ picks is je ROI statistisch betrouwbaar.
HITRATE
Percentage gewonnen weddenschappen.

Let op: een hoge hitrate zegt niet alles. Bij value betting met hogere quotes (2.50+) is 40–45% hitrate al winstgevend op lange termijn.
📡 Data & Technologie
HOE WERKT HET MODEL?
TOTO AI combineert drie lagen:

1. Teamstatistieken — goals voor/tegen thuis en uit, vorm (laatste 5 wedstrijden), min. 8 wedstrijden data vereist

2. Poisson model — wiskundige kansberekening op basis van aanvalskracht en verdedigingszwakte. Per competitie gekalibreerd (Bundesliga offensiever dan Serie A)

3. AI validatie — Claude vergelijkt Poisson kansen met boekmakersquotes en geeft een concreet oordeel met reden
