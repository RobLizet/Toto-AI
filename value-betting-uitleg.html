Wat is value betting?

Value betting draait niet om "winnen we deze ene weddenschap?", maar om structureel weddenschappen aangaan waar je een voordeel hebt. Deze uitleg laat stap voor stap zien wat value is, hoe je een edge berekent en hoe je de getallen leest — met een concreet voorbeeld.

Value in één zin

Value ontstaat wanneer je zelf inschat dat een uitslag vaker gebeurt dan de bookmakers "geloven". Denk je dat een team 24% kans heeft om te winnen, terwijl de faire markt op ~16% rekent, dan is er value — een statistisch voordeel. Het doel is niet één weddenschap winnen, maar over veel picks heen een voordeel pakken. Dat voordeel levert op de lange termijn rendement op.

Modelkans vs. faire marktkans

Om value te vinden vergelijk je twee kansen met elkaar:

  • Modelkans — de kans die je zelf schat. ProMatchXI gebruikt hiervoor een Poisson-model (op basis van doelpunten, vorm en onderlinge duels) plus een AI-analyse.
  • Faire marktkans — de kans die in de odds zit, nádat de bookmaker-marge eruit is gehaald. Bookmakers bakken altijd een winstmarge in hun odds; die halen we eruit ("de-vig") om eerlijk te kunnen vergelijken.

Belangrijk: je vergelijkt niet met één bookmaker, maar met de consensus van veel bookmakers. Dat is een stabielere maatstaf voor wat de markt echt denkt.

Edge = het kerngetal

Edge = modelkans − faire marktkans (in procentpunten). Dit bepaalt of een pick goed genoeg is. Een edge van +8% betekent: het model schat 8 procentpunt meer kans dan de eerlijke markt.

De edge is het selectiegetal. Alleen picks met genoeg edge én genoeg betrouwbaarheid blijven over — dat zijn de picks die je uiteindelijk te zien krijgt.

EV% en Kelly: hoeveel zet je in?

De edge bepaalt óf een pick meedoet. Voor hoeveel je inzet kijk je naar twee andere getallen:

  • EV% — het verwachte rendement op je inzet, op basis van de odds waarop je echt wordt uitbetaald.
  • Kelly — een rekenregel voor een verantwoorde inzetgrootte. ProMatchXI toont een halve Kelly, wat voorzichtiger is dan de volledige.

Een uitgewerkt voorbeeld

Stel: Canada speelt thuis tegen Ierland en je ziet de pick "Ierland wint". Zo lees je de getallen:

GetalWaardeWat het betekent
Kans24%Het model schat 24% kans dat Ierland wint.
Faire marktkans~16%Odds 6.00, marge eruit → markt rekent op ~16%.
Edge+8%24% − 16% = 8 procentpunt voordeel. Dit is de "value".
Odds@6.00De prijs waarop je inzet wordt uitbetaald.
Betrouwbaarheid5/10Weinig data over deze landen → onzekere schatting.

Let op: de edge (+8%) en de betrouwbaarheid (5/10) zijn twee verschillende dingen. Er is voordeel, maar omdat de data dun is ("schaarse data"), is het een zwakkere pick. Een goede tool laat allebei zien, zodat je het zelf kunt afwegen.

Dit is een rekenvoorbeeld ter illustratie, geen echte pick of voorspelling.

Value is geen garantie

Value betting geeft je een voordeel over veel weddenschappen heen — geen zekerheid per stuk. Een enkele weddenschap kun je met value alsnog verliezen; dat hoort erbij. De winst zit in de herhaling: consequent picks nemen waar je een edge hebt.

CLV: je echte scorebord

Het beste bewijs dat je écht voordeel hebt, is Closing Line Value (CLV): versloeg je inkoopkoers de eindkoers (de slotkoers vlak voor de aftrap)? Wie de slotkoers structureel verslaat, verslaat de markt — en dat zegt op de lange termijn meer dan de uitslag van één weddenschap.

In het kort

Value = jouw modelkans hoger dan de faire (marge-vrije) marktkans. Edge selecteert, EV en Kelly bepalen de inzet, en CLV bewijst op termijn of je gelijk had.
